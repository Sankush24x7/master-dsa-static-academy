- HTML5
- CSS3
- Vanilla JavaScript

No framework, no backend, no database, no build tools.  
Just open `index.html` and start learning.

## Core Highlights

- 35 chapter DSA curriculum (Beginner → Interview Mastery)
- Sticky sidebar with phase-based chapter navigation
- Single-expand accordion behavior in phase tabs
- Dark/Light theme toggle with localStorage persistence
- Live chapter search
- Chapter completion tracking and progress dashboard
- Reading progress bar
- Notes system (localStorage)
- Bookmark system (localStorage)
- Daily goal tracker + solved counter
- Motivational streak tracker
- Previous/Next chapter navigation
- Chapter-wise quiz section
- Topic-wise curated direct source links (9 buttons, no Google redirect)
- Fully responsive for desktop and mobile

## Run Locally

```bash
git clone https://github.com/Sankush24x7/master-dsa-static-academy.git
cd master-dsa-static-academy
```

Open `index.html` directly in your browser.

---

## Deploy on GitHub Pages

1. Push code to `main` branch.
2. Open repository settings:
   `https://github.com/Sankush24x7/master-dsa-static-academy/settings/pages`
3. Under **Build and deployment**:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
4. Save and wait 1-2 minutes.

Your site URL:

`https://sankush24x7.github.io/master-dsa-static-academy/`
